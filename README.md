# Open/Closed Principle (OCP)

## Опис
Принцип відкритості/закритості означає, що класи повинні бути
відкритими для розширення, але закритими для змін.

## Приклад
У цьому репозиторії реалізовано систему знижок:
- Discount — базовий клас
- StudentDiscount, VipDiscount — розширення
- DiscountCalculator — працює з будь-якою знижкою без змін коду
